Compiling the Fwink Source Code

Fwink was compiled in the following environment:

  • Visual Studio 2005 Standard Edition
  • Platform SDK for Windows Server 2003 R2
  • DirectX SDK April 2006

I believe that is all you need. I do not know if Visual Studio 2005 Express Edition can load the solution & project files.

MS Layer For Unicode

Fwink is a fully Unicode application. It links with the Microsoft Layer for Unicode (MSLU) in order to run on Windows 98 and ME.
